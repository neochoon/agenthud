# agenthud

CLI tool to monitor agent status in real-time. Works with Claude Code, multi-agent workflows, and any AI agent system.

## Overview

```bash
$ agenthud --watch

┌─ agenthud ──────────────────────────┐
│ Agent: Claude Code                         │
│ Branch: feat/123-auth                      │
│ Plan: 3/5 steps done                       │
├─ Today ────────────────────────────────────┤
│ +3 commits  +142 lines  -23 lines          │
├─ Tests ────────────────────────────────────┤
│ ✓ 12 passed  ✗ 1 failed                    │
└────────────────────────────────────────────┘
```
